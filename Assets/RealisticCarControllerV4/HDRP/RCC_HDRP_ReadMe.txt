This package includes HDRP version of the asset. Import it only if your project is HDRP. 

WARNING
Importing this package will overwrite all assets in the project.

Installation steps

1. Be sure your project has HDRP installed and configured through Window --> Rendering --> HDRP Wizard. Be sure you don't have any errors.
2. Import normal version of the asset from the asset store. It's builtin render pipeline and URP supported version.
2. After the import is done, import this package in the HDRP folder.
3. Importer window will select necessary assets going to be changed.
4. After the import is done, all necessary assets will be changed in the project.
